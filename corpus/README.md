# Corpus Generators

This directory contains the scripts that generate MOSAIC's synthetic benchmark corpus and evaluation suite.

Public repo note:

- generated corpus documents are not meant to be committed
- `mosaic demo --clean --fast` or the generator commands will recreate them locally

Useful commands:

```powershell
.\.venv\Scripts\mosaic.exe generate-corpus --clean
.\.venv\Scripts\mosaic.exe generate-eval
```
